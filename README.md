# Data-Science-Project-Template
This template is designed for end-to-end data science and machine learning projects, including:

- exploratory data analysis (EDA)
- feature engineering and modeling
- data validation (Great Expectations)
- API deployment (FastAPI / Gradio)
- containerization (Docker)
- cloud readiness (AWS / ECS)

It separates local experimentation from production runtime acording to the best practice in professional ML projects.
